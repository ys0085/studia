Opis Projektu

Mastermind to gra polegająca na zgadywaniu czterocyfrowych liczb według pewnych ustalonych zasad. Ten projekt to program w języku C, który poprawnie zgadnie wybraną czterocyfrową liczbę w maksymalnie 7 pytaniach. 



    while(round < 9){
        response = ask(currentGuess);
        if(response == 40){
            solFound = 1;
            break;
        }
        removeComb(pG, currentGuess, response);
        currentGuess[0] = '0';
        for(int i = 0; i < 1296; i++){
            if(pG[4*i] != '0'){
                strncpy(currentGuess, &pG[4*i], 4);
                
            }
        }
        if(currentGuess[0] == '0') break;
        round++;
    }
